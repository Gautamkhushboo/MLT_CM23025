Assignment 3

Demonstrate tensor reshaping and explain the difference between reshape() and flatten().

METHOD 01

reshape()

Change the shape of a tensor without changing its data. Total elements must stay the same.

// Output will appear here... 
// 1D [6] → 2D [2,3]
const t = tf.tensor1d([1,2,3,4,5,6]);
const reshaped = t.reshape([2, 3]);
console.log("Original shape:", t.shape);
console.log("After reshape:", reshaped.shape);
reshaped.print();
METHOD 02

flatten()

Flatten any tensor to a 1D vector. Always results in shape [n].

// Output will appear here... 
// 2D [2,3] → 1D [6]
const mat = tf.tensor2d([[1,2,3],[4,5,6]]);
const flat = mat.flatten();
console.log("Original shape:", mat.shape);
console.log("After flatten:", flat.shape);
flat.print();
